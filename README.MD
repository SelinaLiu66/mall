# 购物商城项目-mall

## 1.项目简介

本项目是基于微服务架构的一个后端购物商城项目，一共五个服务。
包含订单、商品、用户、三个服务。一个公共服务，公共服务包含一些公共的实体类和工具类。
还有网关服务**
